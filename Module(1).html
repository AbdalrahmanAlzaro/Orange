About HTML Formatting

HTML gives us powerful tools to format text. Semantic tags are always preferred over visual-only tags. For example, emphasis adds meaning, not just style. You should never skip the alt attribute on images because it matters for accessibility. The old font tag is deprecated and no longer used in modern HTML. The correct approach is to use CSS for all visual styling instead.

Block vs Inline Elements

This entire box is a block-level div — it starts on its own line and stretches the full width of the page. Inside this paragraph, the colored words are inline span elements — they sit naturally within the text flow without breaking to a new line. That is the core difference: block elements stack vertically, inline elements flow horizontally within content.
